# 系统指令

# 插值语法 v-text、v-html、{{}}

v-text

往标签内部插入普通文本，等价于 {{}} 。默认都不支持代码的解析，可以用来防治XSS攻击。

XSS攻击：XSS攻击通常指的是通过利用网页开发时留下的漏洞，通过巧妙的方法注入恶意指令代码到网页。

<div v-text="str"></div>

v-html

往标签内部插入html文本

<div v-html="str"></div>

{{}}

{{}} 除了可以进行数据的渲染和方法的调用外，还支持一些书写业务逻辑，复杂的业务逻辑不建议书写在内，影响代码的可读性

<!-- 1.支持三元运算符 -->
<div> {{ a > b ? c : d }} </div>

<!-- 2.支持方法调用 -->
<div> {{ 'abc'.split('').reverse().join('') }} </div>

# 属性绑定 v-bind 或 :

html标签中的属性，默认是不能解析变量的（默认不能写vue语法）

我们需要解决这个问题就需要进行属性绑定，属性绑定是单向的，数据只能从data内流向页面

<!-- 1.属性绑定 -->
<a v-bind:href="url"> 小苏同学 </a>

<!-- 2.简写 -->
<a :href="url"> 小苏同学 </a>

<!-- 3.控制类名（对象写法） -->
<div :class="{active:bool}"> 小苏同学 </div>

<!-- 4.控制类名（数组写法） -->
<div :class="['active']"> 小苏同学 </div>

<!-- 5.控制类名（三元写法） -->
<div :class="bool ? 'active':''}"> 小苏同学 </div>

<!-- 6.控制行内样式（对象写法） -->
<div :style="{color:bool?'red':'blue'}"> 小苏同学 </div>

# 事件绑定 v-on 或 @

<button v-on:click="num++">按钮</button>
<button @click="num++">按钮</button>

# 条件渲染 v-if

v-if值为false时，这个节点会被浏览器删除，涉及到DOM操作。

<!-- v-if -->
<div v-if="bool"> 小苏同学 </div>

<!-- v-else -->
<div v-if="bool"> 小苏同学 </div>
<div v-else> 小苏同学 </div>

<!-- v-else-if -->
<div v-if="bool1"> 小苏同学 </div>
<div v-else-if="bool2"> 小苏同学 </div>
<div v-else-if="bool3"> 小苏同学 </div>
<div v-else> 小苏同学 </div>

# 元素显隐 v-show

v-show值为false时，这个节点会被浏览器隐藏，这里的隐藏就是为该元素设置 style 属性，并设置样式为 display:none;

如果频繁操作元素显示不显示，那么就使用 v-show，如果不频繁显示或不显示元素，就使用 v-if

<div v-show="bool"> 小苏同学 </div>

<div style="display:none"> 小苏同学 </div>

# 数据循环 v-for

key是给Vue内部使用的，是一个用于判断节点需不需要进行更新的标识，若key的值没改变则不需要更新节点，反之。

<ul>
    <li v-for="item,index in list" :key="index"> {{ item }}</li>
</ul>

key属性的作用：

  1. 减少不必要的节点更新，提高更新效率（高效更新虚拟DOM）。
  2. 若组件需要重新加载，可以改变key的值。

key值的选择：

  1. key值一般为数字或字符串。
  2. 如果数组不需要修改，key的值写index即可。
  3. 如果数组需要修改，为保证唯一性，一般会写item.id。

不写key属性的隐患：

当选中第一条数据时，此时点击按钮向上添加一条新数据，这时会发现先前选中的数据会变为当前添加的数据。也就是选中的是当前的索引值，并不是数据本身。

<button @click="add">按钮</button>
<ul>
  <li v-for="item in arr">
  	<input type="checkbox"> {{item.name}}
  </li>
</ul>

# 双向数据绑定 v-model

v-model只能用于表单，用于让表单的value值和data中的键进行双向绑定，彼此间互相影响。

# 文本框

<template>
  <div>
    <input type="text" v-model="txtVal" />
    <p>{{ txtVal }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      txtVal: "",
    };
  },
};
</script>

# 斗篷语法 v-cloak

[v-cloak]{
	display: none;
}
</style>

<div v-cloak>
  {{ str }}
</div>
